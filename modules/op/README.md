# 1Password CLI

## Setup

On `macOS` symlink the 1password ssh agent to `~/.config/1password/agent.sock`.

```sh
mkdir -p ~/.config/1password && \
  ln -s ~/Library/Group\ Containers/2BUA8C4S2C.com.1password/t/agent.sock ~/.config/1password/agent.sock
```
