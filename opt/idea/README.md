# IntelliJ IDEA

Setup JetBrains [IntelliJ IDEA](https://www.jetbrains.com/idea/) to use XDG
directories.

## Setup

1. Run `make` from this directory.
2. [Optional] Copy existing IDEA settings to `~/.config/idea/settings`.
